---

## 🚀 Project Overview
This project is a **Machine Learning-based Sentiment Analysis system** that classifies movie reviews using **Natural Language Processing (NLP)** techniques.  
It processes raw text data and predicts sentiment with high accuracy.

---

## 🎯 Objective
Build an efficient model to classify movie reviews and deploy it using a **user-friendly GUI** with accuracy **≥ 95%**.

---

<p>
✔ Python &nbsp;&nbsp; ✔ Pandas & NumPy &nbsp;&nbsp; ✔ Scikit-learn <br>
✔ NLTK &nbsp;&nbsp; ✔ Streamlit
</p>

---

## ⚙️ Key Features
✨ Text preprocessing (cleaning, stopword removal)  
✨ TF-IDF feature extraction  
✨ Multiple ML model comparison  
✨ Best model selection (SVM)  
✨ Interactive GUI for real-time prediction  

---

## 📊 Model Performance

| Model | Accuracy | Precision | Recall | F1 Score |
|------|---------|----------|--------|----------|
| Logistic Regression | ~89% | High | High | Balanced |
| Naive Bayes | ~85% | Medium | High | Moderate |
| SVM | ⭐ **~95%+** | High | High | Best |
